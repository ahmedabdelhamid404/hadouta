# PDF Redesign — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace Sprint 2's functional-but-generic PDF (`hadouta-backend/src/lib/pdf/render-book.ts`) with the cover/body/end-page system designed in `docs/design/specs/2026-05-03-pdf-redesign-spec.md` — three coherent page types, paper texture, three-font hierarchy, ✦ ornament family, watercolor washes — driven by a new `moralStatement` field on the story output.

**Architecture:** Three template helper functions inside `buildHtml()` (cover, body-page, end-page) sharing one CSS block (paper texture, palette, typography, ornaments, watercolor washes). New top-level `moralStatement` field on the Zod story schema, generated by an updated system prompt and seeded in all three few-shot examples. Rendered on the end-page above "النهاية" in Aref Ruqaa. `parentDiscussionQuestion` stays in `storyJson` but is no longer rendered.

**Tech Stack:** TypeScript 5 strict, Zod 3.x, Vercel AI SDK + GPT-4o-mini, Puppeteer 24 (headless Chromium), vitest, Cairo + El Messiri + Aref Ruqaa via Google Fonts.

**Spec:** `docs/design/specs/2026-05-03-pdf-redesign-spec.md`

This is SEPARATE from the moralMoment page's text. The page text shows the moral *embedded in narrative* ("هنا فكرت..."); the moralStatement states the lesson *as a takeaway* ("وعرفت هنا إن...", "وفي الآخر..."). Used on the end-page above "النهاية".

   - Storyteller voice — declarative, warm, parent-to-child register
   - Names the moral concept explicitly (e.g. "التعاون"، "الصدق"، "الشجاعة")

export function buildHtml(input: HtmlInput): string {
  const coverHtml = renderCoverPage({
    title: input.title,
    dedication: input.dedication,
    coverUrl: input.coverUrl,
  });

  const bodyHtml = input.pages
    .map((p) =>
      renderBodyPage({
        pageNumber: p.pageNumber,
        storyText: p.storyText,
        illustrationUrl: p.illustrationUrl,
        moralMoment: p.moralMoment,
      }),
    )
    .join("\n");

  const lastBodyPage = input.pages[input.pages.length - 1];
  const endHtml = renderEndPage({
    moralStatement: input.moralStatement,
    backdropUrl: lastBodyPage?.illustrationUrl ?? input.coverUrl ?? "",
  });

  return `<!doctype html>
<html lang="ar" dir="rtl">
<head>
<meta charset="utf-8" />
<title>${escapeText(input.title)}</title>
<link rel="preconnect" href="https://fonts.googleapis.com" />
